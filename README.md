# Rin Satsuki
Rin Satsuki is a mod that adds (surprise!) Rin Satsuki to Terraria!

![Discord Icon](https://cdn.discordapp.com/attachments/701832459917983747/712088978328912003/discordicon.png)

It currently adds: 
 - 1 town NPC
 - 1 vanity set
 - 15 buffs
 - 1 flask buff
 - 1 debuff
 - 18 potions
 - 1 power-up
